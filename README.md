# FlatMan

An Android application to keep track of your apartments. As a backend it
uses [FlatMan REST API](https://github.com/panosdim/flatman-api)

## Technologies Used

* Kotlin with Coroutines
* Room
* Firebase
* LiveData
* Retrofit
* MPAndroidChart
